# MANTRA Academy Chennai Website

A responsive frontend website developed as part of the MANTRA 2026 Summer School Web Development Assignment.

## Website Type
Educational Institute Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Netlify

## Pages Included
- Home
- About Us
- Courses
- Faculty
- Gallery
- Admissions
- Contact

## Features
- Responsive design for desktop, tablet and mobile
- Clean and modern user interface
- SEO optimization (Title, Meta Description, Meta Keywords, Canonical Tags and Favicon)
- Responsive navigation menu
- Admission form validation using JavaScript
- Contact form
- Faculty section with profile cards
- Course information section
- Campus gallery
- Mobile-friendly layout
- Semantic HTML5 structure

## SEO Features
- Title tag on every page
- Meta description
- Meta keywords
- Image alt attributes
- Canonical tags
- Favicon
- Responsive design

## How to Run

1. Download or clone this repository.
   https://github.com/golivajinsaiseshu-blip/Mantra-Educational-Website-.git
3. Open the project folder.
4. Double-click `index.html` or open it in any modern web browser (Google Chrome, Microsoft Edge, or Mozilla Firefox).
5. Use the navigation menu to explore all pages of the website.
